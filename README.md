# Salla Call Center Performance Dashboard

## Project Overview

This project analyzes call center performance data from Salla using Power BI.
The goal is to monitor agent performance, call volume, response times, and customer service efficiency.

The dashboard provides clear visual insights that help managers understand operational performance and improve customer support quality.

---

## Tools & Technologies

* Microsoft Power BI
* Power Query (Data Cleaning & Transformation)
* Data Visualization
* Excel / CSV Dataset

---

## Data Preparation

The dataset was cleaned and transformed using Power Query. The following steps were performed:

* Removing duplicate records
* Handling missing values
* Formatting date and time columns
* Creating calculated columns for analysis

---

## Dashboard Features

The Power BI dashboard includes several key performance indicators (KPIs):

* Total Calls Received
* Answered vs Missed Calls
* Average Call Duration
* Agent Performance
* Call Volume by Date
* Customer Service Efficiency

---

## Key Insights

* Identify peak call hours and busiest days
* Evaluate individual agent performance
* Monitor unanswered calls
* Track average call handling time

These insights help improve operational efficiency and customer satisfaction.
